## 1. 产品概述

小区门卫室访客车辆临时停放告示贴，用于展示访客车辆停放状态，协助门卫高效管理临时车辆。

- 主要用途：小区门卫室监控访客车辆停放状态，快速识别超时未离场车辆
- 目标用户：小区门卫、安保人员
- 核心价值：提升门卫对临时停放车辆的管理效率，减少超时停放纠纷

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|---------|---------|
| 门卫人员 | 无需注册，直接使用 | 查看停放记录、筛选车牌、登记离场 |

### 2.2 功能模块

1. **数据展示顶栏**：当前在场数、超时数、今日累计离场数
2. **在场车辆列表**：按驶入时间倒序展示，超时条目左侧加叹号标记
3. **已离场折叠区**：默认收起，显示条数，可展开查看
4. **车牌筛选**：输入车牌尾号后两位即时过滤
5. **登记离场**：点击按钮将条目移至已离场区（仅前端状态变更）

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 首页 | 顶栏统计 | 展示当前在场车辆数、超时车辆数、今日累计离场数 |
| 首页 | 车牌筛选 | 输入框输入车牌尾号后两位，实时过滤列表 |
| 首页 | 在场车辆列表 | 卡片式列表，按驶入时间倒序；超时条目左侧显示红色叹号 |
| 首页 | 已离场折叠区 | 默认收起，显示已离场条数；点击展开查看历史记录 |
| 首页 | 登记离场操作 | 每条在场记录右侧按钮，点击后移至已离场区 |

## 3. 核心流程

门卫查看告示贴 → 浏览在场车辆列表 → 如需查找特定车辆，输入车牌尾号后两位筛选 → 发现车辆离场，点击"登记离场" → 条目自动移至已离场折叠区 → 顶栏统计数据实时更新

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深蓝色（#1e3a5f），体现安保专业感
- 强调色：红色（#dc2626）用于超时警示，绿色（#16a34a）用于正常状态
- 字体：思源黑体 + Noto Sans SC，清晰易读
- 卡片式布局，圆角 8px，柔和阴影
- 图标：lucide-react 线性图标，简洁明了

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 首页 | 顶栏统计 | 深蓝背景，白色数字，卡片横向排列，数字醒目 |
| 首页 | 筛选区 | 浅灰背景输入框，左侧搜索图标，placeholder "输入车牌尾号后两位" |
| 首页 | 在场列表 | 每行显示车牌、来访住户、驶入时间、允许停放至、超时状态、操作按钮 |
| 首页 | 已离场区 | 折叠标题带条数标记，展开后灰色调显示历史记录 |
| 首页 | 超时标记 | 红色圆形感叹号图标，位于条目最左侧，视觉突出 |

### 4.3 响应式

- 桌面端优先设计，适配门卫室电脑屏幕
- 卡片列表固定高度，超出滚动
- 顶栏统计自适应宽度

### 4.4 动效设计

- 条目移入已离场区时添加淡出动画
- 折叠区展开/收起有平滑过渡
- 按钮悬停有轻微放大和背景变化
- 页面加载时数字从 0 滚动到目标值
